Example Problems

Example 1: 1D Motion - Car Braking

A car traveling at 30 m/s applies brakes with constant deceleration of 5 m/s². Find:
  1. Time to stop
  2. Stopping distance

Given: v₀ = 30 m/s, v = 0 m/s, a = -5 m/s²

Solution:

1) Using v = v₀ + at:

0 = 30 + (-5)t → t = 6 s

2) Using v² = v₀² + 2aΔx:

0 = 900 + 2(-5)Δx → Δx = 90 m

Answer: 6 seconds to stop, 90 meters stopping distance

Example 2: Projectile Motion - Ball Launch

A ball is launched at 25 m/s at 40° above horizontal. Find:
  1. Maximum height
  2. Range
  3. Time of flight

Given: v₀ = 25 m/s, θ = 40°, g = 9.81 m/s²

Components:

v₀ₓ = 25 cos(40°) = 19.15 m/s

v₀ᵧ = 25 sin(40°) = 16.07 m/s

Solution:

1) Max height: h = v₀ᵧ²/(2g) = 16.07²/(2×9.81) = 13.16 m

2) Flight time: T = 2v₀ᵧ/g = 2×16.07/9.81 = 3.28 s

3) Range: R = v₀ₓ × T = 19.15 × 3.28 = 62.81 m

Answer: Height = 13.2 m, Range = 62.8 m, Time = 3.3 s

Example 3: Circular Motion

A car rounds a curve of radius 50 m at 20 m/s. Find the centripetal acceleration.

Given: r = 50 m, v = 20 m/s

Solution:

Using aₒ = v²/r:

aₒ = 20²/50 = 400/50 = 8 m/s²

Answer: Centripetal acceleration = 8 m/s² toward center

Quick Reference Card

Problem-Solving Strategy:

  1. Identify knowns and unknowns - List all given values and what you need to find
  2. Choose coordinate system - Set positive directions and origin
  3. Select appropriate equations - Pick based on what's missing
  4. Check dimensions - Ensure units are consistent
  5. Verify answer - Does it make physical sense?

Common Mistakes to Avoid:

  • Mixing up displacement (Δx) with position (x)
  • Forgetting that acceleration can be negative
  • Not considering initial conditions (v₀, x₀)
  • Using wrong sign for g in projectile motion (-9.81 m/s²)
  • Confusing average and instantaneous values

Special Cases:

Free Fall v₀ = 0, a = -g
Horizontal Launch θ = 0°, v₀y = 0
Maximum Range θ = 45° (on level ground)
Uniform Motion a = 0, v = constant
